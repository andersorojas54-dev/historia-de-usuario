# Historia de usuario
## 📦 Sistema de Inventario en Python

Este repositorio contiene un sistema de inventario desarrollado en Python, el cual evoluciona desde una versión básica hasta una versión más completa con almacenamiento en archivos JSON.

## 🚀 Descripción

El programa permite gestionar productos mediante un menú interactivo en consola.
El usuario puede agregar productos, visualizar el inventario, calcular estadísticas y guardar o cargar la información.

📁 Archivos del proyecto
	•	Historia_de_usuario_1.py → Versión básica (un producto y cálculo simple)
	•	Historia_de_usuario_2.py → Inventario con múltiples productos y menú
	•	Historia_de_usuario_3.py → Sistema completo con almacenamiento en JSON

## ⚙️ Funcionalidades
•	Agregar productos
•	Mostrar inventario
•	Calcular estadísticas
•	Guardar datos en JSON
•	Cargar datos desde JSON
•	Menú interactivo

## 🧠 Conceptos utilizados
•	Listas y diccionarios
•	Funciones
•	Condicionales (if, elif, else)
•	Bucles (while, for)
•	Manejo de errores (try/except)
•	Archivos JSON

## 📊 Estadísticas

El sistema calcula:
•	Valor total del inventario
•	Cantidad total de productos

Practicar programación en Python mediante el desarrollo progresivo de un sistema de inventario, aplicando estructuras de datos, lógica de programación y almacenamiento de información.
